# HR-Analytics-Dashboard
HR analytics dashboard built using SQL and Power BI  on IBM HR dataset — analyzing employee attrition patterns.
# HR Analytics Dashboard

## Tools Used
MySQL | Power BI

## Dataset
IBM HR Analytics — 1,470 employees
Download from Kaggle:
kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset

## SQL Queries
10 queries analyzing attrition patterns

## Key Findings
- Sales dept highest attrition at 20.63%
- Overtime employees leave 3x more
- Under 25 employees leave most at 34.5%
- Low salary = 3x higher attrition
